CADENAS DE CARACTERES
Son arreglos o vectores con la particularidad de guardar caracteres en sus espacios de memoria, como lo son numeros y letras.

Estas cadenas de caracteres se pueden declarar de las siguienes formas:
char array[]={'a','e','i','o','u'};
char array[]={aeiou};

Al momento de imprimir la variable "array" esta mostrara como resultado aeiou
Para imprimir una cadena de caracteres lo haremos de la siguiente manera:
Si el numero de espacios de memoria es mayor a uno se usara => printf("%s", array);
Si el numero de espacios de memoria es igual a uno se usara => printf("%c", array);


En el lenguaje C existen ciertas funciones que facilitan el control de la cadenas de texto estas son:

  • gets(var)
  • strlen(var)
  • strrev(var)
  • strcpy(var,var2)
  • strcmp(var, var2)
  • strupr(var)
  • strlwr(var)
  • atoi(var)
  • atof(var)

para poder utilizar las anteriores funciones necesitamos la libreria: string.h

gets()
Con esta funcion podemos almacenar una cadena de caracteres dentro de una variable mientras estamos ejecutando el programa
gets(array);

strlen()
Esta funcion retorna el numero de espacios de memoria almacenados por el array
5 = strlen(array);

strrev()
Esta funcion invierte el array ingresado
strrev(array); // => {uoiea}

strcpy(,)
Esta funcion copia el segundo parametro en el primero parametro.
strcpy(array,array2);// => {}
como el segundo parametro está vacio el primero tambien quedara vacio.

strcmp(,)
Esta funcion compara dos cadenas de caracteres, si estas son identicas retornaran el valor de 0 pero si estas son diferentes retornaran el valor de 1.
1 = strcmp(array,array2);

strupr() y strlwr()
Estas funciones convierten una cadena de caracteres a mayusculas o minusculas.
strupr(array); // => {AEIOU}
strlwr(array); // => {aeiou}

atoi() y atof()
Estas funciones retornan un valor entero o un valor flotante, las cadenas de caracteres deben ser numericas
12 = atoi(arrayNumerico);
15.24 = atof(arrayNumerico);

ARREGLOS EN PARALELO
Es una relación entre las componentes de igual subindice (la misma posición) de dos o n vectores. Estos vectores permiten guardar información de manera más ordenada, por lo tanto el usuario pordra entenderla de manera mas rapida.

Ejemplo
En cuatro vectores se encuentra la informacion de cinco personas,en v1 se encuentra el nombre de las personas, en v2 se encuentra su edad, en v3 se encuentra su genero de sexo y en v4 se encuentras su ocupación.

ARREGLOS BIDIMENSIONALES/ MATRICES
Un arreglo bidimensional es aquel que tiene m * n posiciones de memoria, en otras palabras es una tabla de datos.
Donde a m se le podria definir como numero de filas y a n como numero de columnas

Comó declarar una matriz
Sede clara con el siguiente orden:
tipo de dato nombre[m][n];
Para indecar una posicion de memoria primero indicamos el numero de fila y despues su numero de columna.

(1,1)(1,2)(1,3)(1,n-1)(1,n)
(2,1)(2,2)(2,3)(2,n-1)(2,n)
(m-1,1)(m-1,2)(m-1,3)(m-1,n-1)(m-1,n)
(m,1)(m,2)(m,3)(m,n-1)(m,n)


Ejemplo
int matriz[3][2] = { {9,7} , {1,2} , {5,6} }
